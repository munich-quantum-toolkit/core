# MQT Core C++ API {#mainpage}

MQT Core provides the C++ foundations shared across the Munich Quantum Toolkit:
quantum-circuit data structures, decision diagrams, ZX calculus, QIR support,
and device interfaces.

Use the Namespaces and Files entries in the navigation tree to browse the public
API. Search covers all documented entities.

The C++ API links to the version-matched QDMI reference where applicable. QDMI
remains an external project and its documentation is hosted separately.
