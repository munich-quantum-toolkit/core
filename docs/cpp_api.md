# C++ API Reference

The C++ API reference is rendered directly from the source with Doxygen. It is
part of this documentation site and follows the active light or dark color
scheme.

```{button-link} cpp/index.html
:color: primary
:expand:

Open the complete C++ API reference
```

Documentation pages can link to individual C++ entities through the generated
Doxygen symbol inventory. For example, {cpp-api:class}`dd::Package` opens the
corresponding class reference without requiring Sphinx to parse its C++
declaration.

External Doxygen inventories work the same way. The pinned QDMI v1.3.2 inventory
makes links such as {qdmi:func}`QDMI_device_initialize` resolve to the
corresponding versioned QDMI documentation.
